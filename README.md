# shopping-cart-react-2023
This repository contains a practice of a shopping cart using react.

# Getting started
Clone the repository
```git clone https://github.com/edwinb92/shopping-cart-react-2023.git ```

Install packages
```npm install ```

Run project
``` npm run dev```

# eslint and tests

Run unit tests
```npm run test```

Run eslint validations
```npm run lint```
